## Description

As a member of a team, I was involved in creating a plugin for SheetShuttle. The plugin was designed to automate the process of grade management for students. It would take the student grades from a Google Sheet, write the information to a markdown file, and push the file to the student's grade book repository on GitHub. The project was written in Python, which allowed us to leverage the power of the language to accomplish the task efficiently.

Working on this project was a great learning experience for me as I got to use my programming skills to solve a real-world problem. I gained hands-on experience in working with Google Sheets API and GitHub API, which are critical tools for automating various tasks in today's world. I also got to improve my knowledge of Python and learned how to write clean and maintainable code.

In conclusion, the plugin for SheetShuttle project was a challenging and rewarding experience that allowed me to apply my technical skills and knowledge to make a real impact in the field of education.
