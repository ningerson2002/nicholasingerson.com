## Description

A password generator is a tool that creates a secure, random password for a user to use as a login credential. This personal project was created using Python programming language, making use of the built-in string and random modules. The initial implementation was basic, but it was later evolved to include the option of special characters, providing an added layer of security to the generated passwords.

The password generator also has a password manager system that writes the password and the website it's used for in a markdown file called passwords. This makes it easy for users to keep track of all their passwords in one place, rather than having to remember them all or writing them down in a physical notebook. The password manager system also ensures that the passwords are stored securely, as the markdown file can be encrypted if desired.

In conclusion, this personal project is a useful tool for anyone who wants to generate strong and secure passwords for their various online accounts. The addition of the password manager system makes it even more convenient, as users can easily access their passwords whenever they need them. The project demonstrates the ability to create a functional tool using Python and highlights the importance of using secure passwords in today's digital world.
