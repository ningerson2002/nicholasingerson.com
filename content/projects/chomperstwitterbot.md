## Description

For my final project in CS 101, I was part of a team which was involved in creating a Twitter bot that generated haikus. Using Python, I helped design the bot to randomly select words from a dictionary of related terms and arrange them into a haiku poem. The bot, named @chomp_bot, was set to tweet a new, randomly generated haiku every hour. I contributed to the implementation of syllable counting in the code to ensure that the haikus followed the traditional 5-7-5 syllable format. Working with dictionaries in Python was a great learning experience for me, as I got to demonstrate my understanding of data structures in Python. Overall, the project was a fun and challenging opportunity to combine my programming skills with my creativity.
